# Token

## 适用范围

ERC20 / ERC721 / ERC1155 及项目治理或效用代币。不含 AMM/Vault 业务流程（见对应 `products/` 文件）。

## 设计检查清单

- 明确代币标准与是否可升级；优先不可升级 + 可治理参数
- `mint` / `burn` 权限、上限、时间窗；初始供应与分配可审计
- 税费、黑名单、暂停、转账限制：默认最小权限，变更可追踪
- `permit` / EIP-2612：域分隔、nonce、过期；防重放
- 若同一代币用于质押奖励或 LP，说明通胀与激励模型
- 外部 hook 或回调（ERC777 等）的重入与 CEI

## 权限与资金

- 铸币/销毁仅授权角色；紧急暂停有解除路径
- 金库/国库地址变更受 timelock 或多签（若适用）

## 外部依赖

- ERC 交互见 `security.md`；若涉及 swap/预言机，同时遵循 `defi.md`

## 测试要点

- 上限、权限、暂停、permit 失败路径
- 非标准 ERC20（无返回值、fee）若支持，单独用例
